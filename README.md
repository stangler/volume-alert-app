# 出来高急増アラート

TradingView用のPineスクリプト（v6）です。指定した取引時間内に出来高が平均の一定倍率を超えた場合にアラートを発火します。寄り付き・大引けのノイズ時間帯を除外することで、誤検知を抑えた精度の高いアラートを実現しています。

---

## 機能

- 過去N本の平均出来高をもとに、現在の出来高が急増しているかをリアルタイムで判定
- バー確定前のティックでもアラートを発火（`alert.freq_all`）
- 同一バー内での連続アラートを防ぐフィルター機能
- 午前・午後の取引セッションを個別に設定可能
- 寄り付き・大引けのノイズ時間帯を平均計算から除外
- チャート上にラベル・背景色で視覚的に表示
- 現在の状態をテーブルで右上に表示

---

## ノイズ除外の仕組み

### 寄り付きノイズ（除外対象）

| ケース | 対応 |
|---|---|
| 通常寄り付き（9:00〜9:03） | 時刻で固定除外 |
| 大幅上昇・下落による遅延寄り付き | 前足の出来高が0のバーを動的に除外 |

寄り付き直後は出来高が異常に多くなるため、平均出来高の計算から除外しています。また大幅上昇・下落の際は9:03や9:04など時刻がずれて寄り付く場合があるため、「前足の出来高が0→現在足で出来高が発生」を動的に検出して対応しています。

### 大引けノイズ（除外対象）

- **15:25〜15:30** を平均計算・アラート発火の両方から除外
- 大引け前後は出来高が膨らみやすく、誤検知の原因になるため除外しています

ノイズ除外中の時間帯はチャート上で**グレーの背景色**で表示されます。

## パラメータ設定

インジケーター名の横にある ⚙️ アイコン（ペインにホバーすると表示）をクリックすると、以下の項目をGUIから変更できます。

### 基本設定

| パラメータ | デフォルト | 説明 |
|---|---|---|
| 出来高倍率 | 2.0 | 平均出来高の何倍で急増と判定するか |
| 平均出来高の計算期間 | 20 | 過去何本のローソク足で平均を計算するか |

### 取引時間設定

| パラメータ | デフォルト | 説明 |
|---|---|---|
| 午前セッションを有効 | ON | 午前セッションのアラートを有効にするか |
| 午前開始時間 | 9:04 | 寄り付き直後のノイズを避けるため9:04スタート |
| 午前終了時間 | 11:30 | 午前セッションの終了時刻 |
| 午後セッションを有効 | ON | 午後セッションのアラートを有効にするか |
| 午後開始時間 | 12:30 | 午後セッションの開始時刻 |
| 午後終了時間 | 15:25 | 大引けノイズを避けるため15:25で終了 |

---

## アラートの設定方法

1. インジケーターをチャートに追加
2. TradingViewの **「アラート」** ボタンをクリック
3. 条件に **「出来高急増アラート」** を選択
4. 通知方法（プッシュ通知・メール等）を設定して保存

---

## 表示の見方

| 表示 | 意味 |
|---|---|
| 赤い棒 | 前足より出来高が増加 |
| 白い棒 | 前足より出来高が減少 |
| 黄色い線 | 平均出来高 |
| 赤い背景 | 出来高急増を検出中 |
| グレーの背景 | ノイズ除外時間帯（寄り付き・大引け） |
| ラベル（例: 3.25倍） | 急増検出時の出来高比率 |

### テーブル（右上）

| 項目 | 内容 |
|---|---|
| 現在出来高 | 現在バーの出来高 |
| 平均出来高 | 過去N本の平均出来高 |
| 出来高比率 | 現在出来高 ÷ 平均出来高 |
| ノイズ除外中 | 寄り付き・大引けの除外中かどうか |
| 急増検出 | 現在急増を検出しているかどうか |

## 動作環境

- Pine Script v6
- タイムゾーン: GMT+9（日本時間）
