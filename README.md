# 出来高急増アラート

TradingView用のPineスクリプト（v6）です。指定した取引時間内に出来高が平均の一定倍率を超えた場合にアラートを発火します。

---

## 機能

- 過去N本の平均出来高をもとに、現在の出来高が急増しているかをリアルタイムで判定
- 午前・午後の取引セッションを個別に設定可能
- バー確定前のティックでもアラートを発火（`alert.freq_all`）
- 同一バー内での連続アラートを防ぐフィルター機能
- チャート上にラベル・背景色で視覚的に表示
- 現在の状態をテーブルで右上に表示

---

## パラメータ設定

インジケーター名の横にある ⚙️ アイコン（ペインにホバーすると表示）をクリックすると、以下の項目をGUIから変更できます。

### 基本設定

| パラメータ | デフォルト | 説明 |
|---|---|---|
| 出来高倍率 | 2.0 | 平均出来高の何倍で急増と判定するか |
| 平均出来高の計算期間 | 20 | 過去何本のローソク足で平均を計算するか |

### 取引時間設定

| パラメータ | デフォルト | 説明 |
|---|---|---|
| 午前セッションを有効 | ON | 午前セッションのアラートを有効にするか |
| 午前開始時間 | 9:04 | 午前セッションの開始時刻 |
| 午前終了時間 | 11:30 | 午前セッションの終了時刻 |
| 午後セッションを有効 | ON | 午後セッションのアラートを有効にするか |
| 午後開始時間 | 12:30 | 午後セッションの開始時刻 |
| 午後終了時間 | 15:00 | 午後セッションの終了時刻 |

> 午前開始を9:04にしているのは、寄り付き直後（9:00〜9:03頃）はどの銘柄も出来高が多くアラートが頻発するのを防ぐためです。

---

## アラートの設定方法

1. インジケーターをチャートに追加
2. TradingViewの **「アラート」** ボタンをクリック
3. 条件に **「出来高急増アラート」** を選択
4. 通知方法（プッシュ通知・メール等）を設定して保存

---

## 表示の見方

| 表示 | 意味 |
|---|---|
| 赤い棒 | 前足より出来高が増加 |
| 白い棒 | 前足より出来高が減少 |
| 黄色い線 | 平均出来高 |
| 赤い背景 | 出来高急増を検出中 |
| ラベル（例: 3.25倍） | 急増検出時の出来高比率 |

---

## 動作環境

- TradingView（無料プラン以上）
- Pine Script v6
- タイムゾーン: GMT+9（日本時間）
