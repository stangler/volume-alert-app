# 出来高急増アラート — TradingView Pineスクリプト (v6)

日本株の1分足チャートで、出来高が平均の一定倍率を超えた際にリアルタイムでアラートを発火するインジケーターです。

---

## 機能概要

- 過去N本の平均出来高に対して、現在の出来高が指定倍率を超えた際にアラートを発火
- 取引時間外・寄り付き・大引けのノイズを自動除外
- ローソク足の方向（上昇／下落）でアラートを絞り込み可能
- チャート上にリアルタイムで情報テーブルを表示

---

## 設定パラメータ

### 基本設定

| パラメータ | デフォルト | 説明 |
|---|---|---|
| 出来高倍率 | 2.0 | 平均出来高の何倍で急増と判定するか（1.0〜10.0） |
| 平均出来高の計算期間 | 20 | 過去何本のローソク足で平均を計算するか（5〜100） |
| アラート方向フィルター | 両方 | `両方` / `上昇のみ` / `下落のみ` |

> **方向判定の基準：** 現在足の `close > open` を上昇、`close < open` を下落として判定します。`close == open` の足（同値）は「横ばい」扱いとなり、「上昇のみ」「下落のみ」設定時はアラートが出ません。

### 取引時間設定

| セッション | デフォルト | 備考 |
|---|---|---|
| 午前セッション | 9:04〜11:30 | 寄り付き直後のノイズを避けるため9:04スタート |
| 午後セッション | 12:30〜15:25 | 大引けノイズを避けるため15:25で終了 |

---

## ノイズ除外ロジック

以下のバーは平均出来高の計算から除外されます（アラートも発火しません）。

| ノイズ種別 | 条件 | 目的 |
|---|---|---|
| 寄り付きノイズ（時刻固定） | 9:00〜9:03 | 通常の寄り付き急騰を除外 |
| 寄り付きノイズ（動的検出） | 前足の出来高が0で現在足に出来高が発生したバー | 大幅上昇・下落による遅延寄り付きに対応 |
| 大引けノイズ | 15:25〜15:30 | 大引けの異常出来高を除外 |

---

## チャート表示

- **赤い棒グラフ：** 出来高増加中のバー
- **黄色い折れ線：** 平均出来高
- **グレーの折れ線：** 出来高比率（倍率）
- **赤い背景：** 出来高急増を検出したバー
- **グレーの背景：** ノイズ除外時間帯
- **ラベル：** 急増検出時に倍率を表示

### 情報テーブル（右上）

| 項目 | 内容 |
|---|---|
| 現在出来高 | 現在バーの出来高 |
| 平均出来高 | 過去N本の平均 |
| 出来高比率 | 現在出来高 ÷ 平均出来高 |
| 足の方向 | ↑上昇 / ↓下落 / →横ばい |
| 方向フィルター | 現在の設定値 |
| ノイズ除外中 | 除外中かどうか |
| 急増検出 | 現在バーで検出しているか |

## アラートの設定方法

1. チャートにインジケーターを追加し、設定パラメータを調整する
2. チャート上で右クリック →「アラートを追加」またはアラートアイコンをクリック
3. 条件欄で本インジケーターを選択
4. アラート名欄に `{{ticker}}` と手動入力すると、発火時に銘柄コードがアラート名に表示される（TradingViewのプレースホルダー機能）
5. 通知方法（プッシュ通知・メール等）を設定して保存

> **注意：** スクリプトを更新・変更した場合は、既存のアラートを一度削除してから再作成してください。古いアラートは更新前のコードのまま動き続けます。

### アラートメッセージの例

```
7203 / トヨタ自動車
出来高が急増しました！ ↑上昇
出来高: 152300
平均出来高: 48200
倍率: 3.16倍
```

---

## 銘柄ごとに設定を変える方法

TradingViewのインジケーター設定は**チャートごと**に保存されます。銘柄によって設定を変えたい場合は、**銘柄ごとにチャートタブを分けて**それぞれで設定・アラートを作成してください。

**例：**
- タブ1：157A → 方向フィルター「上昇のみ」
- タブ2：9130 → 方向フィルター「両方」

アラートはバックグラウンドで動作するため、タブを開いたままブラウザを最小化していても通知は届きます。ただし**ブラウザを閉じるとアラートが止まる**場合があるため注意してください。
